Interstellar but with a new name!
And More Games!

To install: Fork this project and make a codespace.
Then open a terminal in the codespace.
Type or copy this code in the terminal: pnpm start pnpm install. Then enter.
Then there should be a pop-up in the bottom left corner with an option to make public. Click It.
Go to the option "Ports"
Control Click the link, and then you can type any url and it is unblocked!
